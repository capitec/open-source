Theme Support

All Omni Components support our theme CSS variables

Theme variables are used to style the overall theme of all Omni Components. Each component also can still be styled individually using their own component specific CSS variables, which would override the theme if specified.

Theme variables typically affect:

    ◉ Shared colors

    ◉ Fonts

    ◉ Borders

    ◉ Accessibility options

There are built-in themes available to test how theme variables can affect te various components. Use the theme dropdown to select a built-in theme or select "Light Theme" to view the components in their default state. The source for the built-in themes are available below.

Alternatively, a custom theme can be authored or uploaded to have a live view of its effects on the components.

Try it now by selecting your current theme to be "Custom Theme"!
